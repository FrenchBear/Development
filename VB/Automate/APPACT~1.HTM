AppActivate

Retour à l'aide générale

La commande AppActivate permet d'activer une application afin de lui envoyer une séquence de touches

Syntaxe

AppActivate NomApp

La commande AppActivate accepte l'argument suivant :

Argument

Description

NomApp

Expression de type chaîne indiquant le nom complet ou le début du nom inscrit dans la barre de titre de l'application à activer.

Exemple

AppActivate "42c_a.sit"
Active l'application Talkman émulant 42C et dont la barre de titre comporte "42c_a.sit - Talkman"

Remarques

En activant systématiquement une application avant de lui envoyer une séquence de touches, vous éviterez que cette dernière soit dirigée vers un autre programme ouvert qui pourrait l'interpréter différemment.
L'application active n'est pas forcément celle qui occupe le plein écran. Elle peut être activée même sous forme d'icône ou de bouton.

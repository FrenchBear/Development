MsgBox

Retour à l'aide générale

La commande MsgBox affiche un message dans une boîte de dialogue et attend que l'utilisateur clique sur le bouton [OK] avant de continuer l'exécution du scénario.

Syntaxe : MsgBox Texte [, Réponse]

La syntaxe de l'instruction MsgBox comprend les arguments suivants :

Élément

Description

Texte

Valeur de type chaîne représentant le texte du message à afficher

Réponse

Paramètre facultatif du Visual Basic représentant le nombre et le type de boutons de la boîte de dialogue :
Le paramètre par défaut est : vbOKOnly

Exemple
MsgBox "Ceci est un message !"
Le scénario s'arrête et affiche la boite de dialogue suivante. Il ne poursuivra son exécution que lorsque l'utilisateur aura cliqué sur le bouton [OK].

 

 

 

Remarque
La commande MsgBox laisse le scénario en suspend tant qu'aucune action n'est faite par l'utilisateur sur la boite de dialogue. Néanmoins, elle n'influe pas le déroulement d'autres programmes, même s'il ont été précédemment lancés par Automate.

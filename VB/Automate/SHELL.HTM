La commande Shell permet de lancer une application (dont l'extension est .exe, .bat ou .com)
Elle autorise, après le nom de l'exécutable, le nom d'un fichier reconnaissable par l'application ouverte

Syntaxe : Shell nomfichier[, affichage]

La commande Shell accepte les arguments suivants :

Argument

Description

Nomprog

Argument de type chaîne indiquant le nom et le chemin complet du programme.
Le programme doit impérativement être un .exe, un .bat ou un .com. Il peut être suivi d'un fichier dérivé de l'application, séparé par un espace, mais l'ensemble doit constituer une seul argument.
Par ailleurs, si le chemin complet du programme ou du fichier comporte des noms avec le caractère {ESPACE}, il doit être encadré de doubles guillemets {""}

Taille

Mode d'affichage de la fenêtre
0 =
1 =
2 =
3 =
4 =
5 =

Constante

Valeur

Description

vbHide

0

La fenêtre est masquée et activée.

vbNormalFocus

1

La fenêtre est activée et rétablie à sa taille et à sa position d'origine.

vbMinimizedFocus

2

La fenêtre est affichée sous forme d'icône et activée.

vbMaximizedFocus

3

La fenêtre est agrandie et activée.

vbNormalNoFocus

4

La fenêtre est rétablie à sa taille et à sa position les plus récentes. La fenêtre active reste active.

vbMinimizedNoFocus

6

La fenêtre est affichée sous forme d'icône. La fenêtre active reste active.



Exemples

Shell "C:\TALK32\Talkman.exe"
Lance l'application Talkman

Shell "C:\TALK32\Talkman.exe C:\TALK32\SITES\42c_a.sit"
Lance l'application Talkman et ouvre le fichier 42c_a.sit pour une connexion à 42C

Shell """C:\Program Files\Microsoft Office\Office\Winword.exe"" ""D:\CLORO\Scénarii Automate\Automate.doc"""
Lance l'application Word et ouvre le fichier Automate.doc

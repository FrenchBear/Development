Interpréteur de fichiers de commandes .CLO

 

Automate permet d'exécuter des scénarii d'envois séquentiels de touches ou de commandes à partir de fichiers texte courants.
Il peut ouvrir ou fermer des applications ou des fichiers, simuler une succession de frappes au clavier pour exécuter certaines commandes, paramétrer des champs date ou lancer d'autres procédures automatiques
Automate est particulièrement recommandé pour la planification en différé de tâches longues et répétitives.

Automate accepte tous les fichiers au format texte et interprète les commandes suivantes :

Shell

Exécute un programme

Execute

Ouvre un fichier

Pause

Insère une pause

SendKeys

Envoi de touches

AppActivate

Active un application

Date

Insère une date

MsgBox

Insère une boîte de message

Close

Termine l'application Automate

L'extension spécifique .clo donnée aux fichiers destinés à être exécutés par Automate a pour but de les distinguer des autres fichiers de type texte. Par ailleurs, il est possible d'associer cette extension à l'exécution du programme (Explorateur : Affichage/Options des dossiers/Type de fichiers).

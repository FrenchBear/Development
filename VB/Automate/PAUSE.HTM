Pause

Retour à l'aide générale

La commande Pause suspend le déroulement du scénario pendant une période donnée afin qu'une autre application puisse exécuter ses propres actions.

Syntaxe : Pause Durée

La syntaxe de l'instruction Pause comprend l'argument suivant :

Élément

Description

Durée

Valeur de type réel double représentant la durée de la pause en secondes

Exemple

Pause 10
Le programme s'arrête pendant 10 secondes avant l'instruction suivante

Remarques

La durée est exprimée en secondes ou en fractions de seconde. Dans ce cas, le séparateur décimal doit impérativement être le point(.).

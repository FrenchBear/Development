Execute

La commande Execute permet de lancer une application ou un fichier
Elle autorise, après le nom de l'exécutable, le nom d'un fichier reconnaissable par l'application ouverte

Syntaxe : Execute nomfichier

La commande Execute accepte l'argument suivant :

Argument

Description

Nomprog

Argument de type chaîne indiquant le nom et le chemin complet du programme ou du fichier.
Un programme peut être suivi d'un fichier dérivé de l'application, séparé par un espace, mais l'ensemble doit constituer une seul argument.
Par ailleurs, si le chemin complet du programme ou du fichier comporte des noms avec le caractère {ESPACE}, il doit être encadré de doubles guillemets {""}


Exemples
Execute "C:\TALK32\Talkman.exe"
Lance l'application Talkman

Execute "C:\TALK32\SITES\42c_a.sit"
Ouvre le fichier 42c_a.sit (associé à l'application Talkman) pour une connexion à 42C

Shell """C:\Program Files\Microsoft Office\Office\Winword.exe"" ""D:\CLORO\Scénarii Automate\Automate.doc"""
Lance l'application Word et ouvre le fichier Automate.doc

Shell """D:\CLORO\Scénarii Automate\Automate.doc"""
Ouvre le fichier Automate.doc (associé à Word)

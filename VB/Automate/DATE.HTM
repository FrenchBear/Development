Date

Retour à l'aide générale

La commande Date envoie à l'application active l'équivalent de la frappe au clavier d'une date.

Syntaxe : Date [Format][, Décalage]

La syntaxe de l'instruction Date comprend les arguments suivants :

Élément

Description

Format

Expression de type chaîne indiquant le format de date attendu par l'application active. Cet argument utilise les symboles anglo-saxons : "d" pour le jour, "m" pour le mois et "y" pour l'année.
Si cet argument est laissé vide, Automate utilisera le format par défaut qui est "dd/mm/yyyy"

Décalage

Valeur de type entier indiquant le décalage en jours (positif ou négatif) par rapport à la date du système.
Si cet argument est laissé vide, Date retournera la date du système.

Exemples
Date "dd/mm/yy", -2
retourne 08/05/99 (si la commande est exécuté le 10/05/99)

Date "dddd d mmmm yyyy"
retourne Mardi 10 mai 1999 (si la commande est exécuté le 10/05/99)

Remarque
Même si la commande Date utilise les termes anglo-saxons (d-m-y) elle conserve l'ordre français Jour-Mois-Année.
